get all match key from array within matcher array.

match(Array $source, Array $match, Boolean $allow_null = true, Boolean $all_variable = false, Mixed $default = null) : Mixed
static

Example: $source = array( 'a'=>'A', 'b'=>'B', 'c'=>null, 'd'=>false );

 $matchs = \Roulette\Iterator::match(
     $source,
     array('a','b')
 ); // return: array('a'=>'B','b'=>'B')

 $matchs = \Roulette\Iterator::match(
     $source,
     array('a','b','c','d')
 ); // return: array('a'=>'B','b'=>'B','d'=>false)

 $matchs = \Roulette\Iterator::match(
     $source,
     array('a','b','c','d'), true
 ); // return: array('a'=>'B','b'=>'B','c'=>null,'d'=>false)

 $matchs = \Roulette\Iterator::match(
     $source,
     array('h')
 ); // return: array()

 $matchs = \Roulette\Iterator::match(
     $source,
     array('h'), true, true
 ); // return: array('h'=>null)

 $matchs = \Roulette\Iterator::match(
     $source,
     array('h'), true, true, 5
 ); // return: array('h'=>5)

Arguments

$source

Array

Source array that will be get the match

$match

Array

Array list of match key to fetch

$allow_null

Boolean

Set false to ignored an item in source if value is null, default:true

$all_variable

Boolean

Set true to give a default value if key doesnt exist, default:false

$default

Mixed

Default value to give in property doesn exist, it only used if $all_variable is true, default:null

the callback return
